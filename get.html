Get Metalama

Add the Metalama packages to your project.

dotnet add package
Package List

Optionally, install the Visual Studio extension to get the best coding experience.

  • Diff source code with transformed code.
  • Code Lens with aspect info.
  • Syntax highlighting of aspect code.
  • Aspect Explorer.

The extension is free for individuals, non-commercial use, open-source contributors, and companies with 3 or fewer Metalama users under the Metalama Community license.

For others, a 45-day trial is available.
